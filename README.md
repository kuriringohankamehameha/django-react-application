# Django-React-Redux Full Stack Application

## Requirements
* Python (Pipenv)
* Node (npm)

* To download all Javascript packages, clone this repository and go to this directory, and type:
```
npm install
```

Install all Python dependencies using `pipenv`:
```
pipenv install
```

To spawn a shell in the virtual environment, use:
```
pipenv shell
```

# Run the program
* Running the `npm` webpack scripts:
```
npm run dev
```
* Django Backend Server (Both `npm` and `python` must be running at the same time!)
```
python manage.py runserver
```
